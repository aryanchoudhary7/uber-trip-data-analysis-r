🚖 Uber Trip Data Analysis using R

📌 Overview

This project focuses on analyzing Uber trip data using R to uncover patterns, trends, and insights related to ride demand. The analysis includes data cleaning, exploration, and visualization to better understand how Uber usage varies across time and other factors.

🎯 Objectives
Analyze Uber trip data to identify peak hours and demand trends
Understand ride frequency patterns
Perform exploratory data analysis (EDA)
Create meaningful visualizations for insights
🛠️ Tech Stack
Language: R
Libraries:
ggplot2 – for data visualization
dplyr – for data manipulation
lubridate – for date-time processing
📂 Dataset
Uber trip dataset (contains date, time, and trip details)
Data source: Kaggle
🔍 Analysis Performed
Data cleaning and preprocessing
Extraction of time-based features (hour, day, month)
Analysis of:
Trips per hour
Trips per day
Monthly trends
Visualization of ride demand patterns
📊 Key Insights
Identified peak demand hours during the day
Observed higher ride frequency on specific days
Detected usage trends across months
Visualized patterns that can help in demand prediction

Trips by Day
